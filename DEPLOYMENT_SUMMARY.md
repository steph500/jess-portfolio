## 📝 How to Update Content

### Text Changes
Edit the respective component files in `src/components/`:
- Hero text → `Hero.tsx`
- About info → `About.tsx`
- Skills → `Skills.tsx`
- Experience → `Experience.tsx`
- Certifications → `Certifications.tsx`
- Contact info → `Contact.tsx`

### Color Changes
Edit `src/app/globals.css`:
- Modify gradient colors in `.gradient-text`
- Update colors in `::webkit-scrollbar-thumb`

### Build & Deploy
```bash
npm run build
cp -r out docs
git add docs/
git commit -m "Update content"
git push
```

## 📦 CV Integration

The portfolio includes a "Download CV" button in the Hero section. Currently, it links to `#cv` as a placeholder. To add the actual CV:

1. Upload the CV PDF file to the `public/` folder
2. Update the link in `src/components/Hero.tsx`:
   ```tsx
   <a href="/jess-cv.pdf" download>
   ```
3. Commit and push the changes
